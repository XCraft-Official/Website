# 常见问题

这里收集了玩家最常问的问题，如果您有其他问题，欢迎在 QQ群 或 Discord 咨询。

## 🔌 连接问题

### 如何加入服务器？

1. 确保您使用的是 Minecraft Java版
2. 在多人游戏中点击"添加服务器"
3. 输入服务器地址（请查看首页服务器状态卡片）
4. 点击"完成"后即可加入

### 无法连接服务器怎么办？

::: details 点击展开解决方案

**检查网络连接**
- 确保您的网络连接正常
- 尝试重启路由器或切换网络

**检查服务器状态**
- 查看首页服务器状态卡片确认服务器是否在线
- 关注官方群公告了解维护信息

**检查游戏版本**
- 确保游戏版本与服务器版本一致
- 服务器支持的版本请查看首页信息

**其他解决方案**
- 尝试使用 VPN 或加速器
- 清除 DNS 缓存（命令：`ipconfig /flushdns`）
:::

### 显示"无效会话"怎么办？

这通常是因为您未正确登录 Minecraft 账户：

1. 退出游戏重新登录
2. 检查是否使用了正版账户
3. 如果使用第三方启动器，请确保账户配置正确

## 🎮 游戏问题

### 如何设置家（home）？

| 命令 | 说明 |
|------|------|
| `/sethome <名称>` | 设置一个家 |
| `/home <名称>` | 传送到家 |
| `/delhome <名称>` | 删除家 |
| `/homes` | 查看所有家 |

### 如何保护我的建筑？

服务器提供多种保护方式：

- **领地保护**: 使用木斧选择区域后输入领地命令
- **箱子保护**: 放置箱子时自动保护
- **信任玩家**: 使用信任命令让其他玩家访问您的区域

## 💎 赞助问题

### 赞助后可以获得特权吗？

::: warning 重要说明
根据 Minecraft EULA 规定，服务器不得向玩家出售游戏内特权。赞助为**纯自愿捐赠**性质，不会获得任何游戏内优势。感谢您的支持！
:::

### 可以退款吗？

::: warning 注意
由于赞助为自愿捐赠性质，原则上不支持退款，请谨慎考虑后再进行赞助。
:::

## 🛠️ 技术问题

### 游戏卡顿怎么办？

1. **降低画质设置** - 关闭光影、降低渲染距离
2. **分配更多内存** - 在启动器中调整内存分配
3. **使用优化模组** - 安装 Sodium、Lithium 等优化模组
4. **检查网络延迟** - 使用加速器降低延迟

### 如何安装资源包？

1. 下载服务器推荐的资源包
2. 打开游戏设置 → 资源包 → 打开资源包文件夹
3. 将下载的资源包放入文件夹
4. 在游戏中启用资源包
