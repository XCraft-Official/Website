# 游戏规则

为了维护良好的游戏环境，请所有玩家遵守以下规则。违反规则可能会导致警告、禁言、封禁等处罚。

## ⚠️ 通用规则

### 禁止作弊

- ❌ 禁止使用任何作弊客户端、外挂、脚本（无政府生存模式除外）
- ❌ 禁止利用游戏漏洞获取不正当利益（无政府生存模式除外）
- ❌ 禁止使用自动点击器、宏按键等辅助工具（无政府生存模式除外）
- ❌ 禁止使用透视、飞行等作弊功能（无政府生存模式除外）

### 文明游戏

- ❌ 禁止辱骂、人身攻击其他玩家
- ❌ 禁止发布涉政、涉黄、涉暴等违规内容
- ❌ 禁止恶意刷屏、广告宣传
- ❌ 禁止冒充管理员或官方人员

### 公平竞争

- ❌ 禁止恶意破坏其他玩家的建筑（无政府生存模式除外）
- ❌ 禁止偷窃其他玩家的物品（无政府生存模式除外）
- ❌ 禁止恶意PK（在禁止PVP的区域，无政府生存模式除外）

## 🎮 各模式规则

### 生存模式

| 规则 | 说明 |
|------|------|
| 禁止卡BUG | 发现BUG请及时上报，利用BUG将受处罚 |
| 红石限制 | 大型红石机器请咨询管理员 |
| 刷怪塔 | 允许建造，但不得造成服务器卡顿 |
| 交易 | 玩家间交易请使用官方交易系统 |

### 小游戏模式

| 规则 | 说明 |
|------|------|
| 公平竞技 | 禁止任何形式的不公平竞争 |
| 团队合作 | 团队模式禁止恶意伤害队友 |
| 积分规则 | 禁止刷分行为 |

### 无政府生存

::: info 无政府模式说明
无政府生存模式是一个高度自由的游戏环境，我们尊重玩家的游戏方式选择。

**允许的行为：**
- ✅ 使用作弊客户端、外挂、辅助工具
- ✅ PVP、掠夺、破坏其他玩家建筑
- ✅ 使用脚本、宏按键等自动化工具

**禁止的行为：**
- ❌ 故意造成服务器卡顿或崩溃（卡服）
- ❌ 利用漏洞复制物品导致经济崩溃
- ❌ 发布违法违规内容（涉政、涉黄、涉暴等）
- ❌ 进行违反 Minecraft EULA 的行为

::: warning 重要提示
无政府模式并非"法外之地"，违反上述禁止条款仍将受到封禁处罚。
:::

## 📋 处罚机制

| 违规程度 | 处罚方式 |
|---------|---------|
| 轻微违规 | 警告提醒 |
| 一般违规 | 禁言 1-7 天 |
| 严重违规 | 封禁 7-30 天 |
| 极其严重 | 永久封禁 |

## 📢 举报与申诉

### 如何举报

如果您发现其他玩家违规，请通过以下方式举报：

1. 游戏内使用 `/report <玩家名> <原因>` 命令
2. 在 QQ群/Discord 联系管理员
3. 在 GitHub 提交 Issue

### 如何申诉

如果您认为处罚不当，可以：

1. 在 QQ群/Discord 联系管理员说明情况
2. 提供相关证据证明您的清白

## 📜 规则更新

服务器规则可能会不定期更新，请关注官方公告获取最新信息。

::: tip 提示
如有任何疑问，欢迎在 QQ群 或 Discord 咨询管理员。
